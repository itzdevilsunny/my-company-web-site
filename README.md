# Zorvia Digital

This project is a premium portfolio website for Zorvia Digital, built with React, Vite, and modern web technologies.

## Technology Stack

- **Frontend**: React + Vite
- **Styling**: Vanilla CSS with futuristic glassmorphism and animations
- **Backend**: Express (for OTP and services)
- **Database**: Supabase
- **AI**: Mistral AI integration

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

The project is configured to be pushed to multiple remotes, including the latest at `https://github.com/itzdevilsunny/my-company-web-site.git`.
